## start

```base
yarn start
// or
yarn start:modile
```

## build

```base
yarn build
// or
yarn build:modile
```


## css

```html
<!-- build:css css/index.css -->
<link rel="stylesheet" href="./css/reset.css">
<link rel="stylesheet" href="./css/index.css">
<!-- endbuild -->

// output  css/index.css
```

## js

```html
<!-- build:js js/index.js -->
<script src="./js/index.js" charset="utf-8"></script>
<!-- endbuild -->

// output js/index.js
```
